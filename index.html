Introduction

CSS which stands for "Cascading Style Sheets" describes how HTML elements are to be displayed on the screen. It is a simple design language intended to simplify the process of making web pages presentable.

CSS controls the layout of multiple web pages all at once using external stylesheets. It emphasizes the style of a web page. CSS gives aesthetic to your web pages.

Selectors are used in CSS to specify HTML elements. It is written before curly bracket. The information inside the curly brackets is called a declaration. Declaration containes properties and values that are applied to the selector.

Different properties used in CSS

  • Page Layouts
  • Colors
  • Fonts
  • Tables
  • Grids
  • Images

In this technical documentation, we would be focusing with basic CSS text documentation. This includes TEXT COLOR, TEXT ALIGNMENT, TEXT DECORATION, TEXT TRANSFORMATION, TEXT SPACING and TEXT SHADOW.

Text color

The color property is used to set the color of the text

  • color name such as "blue"
  • HEX value such as "#000000"
  • RGB value such as"rgb(255, 240, 120)"

Text Alignment

The text-align property is used to set the horizontal alignment of a text

There are 4 ways to align a text
  • center
  • left
  • right
  • justify
h1 { text-align: center; } h2 { text-align: left; } h3 { text-align: right; } div { text-align: justify; }

Use these codes to align all of your texts the way you like it to appear!

For vertical alignment of an element (top, middle, bottom) img.top { vertical-align: top; }

The text-decoration property is used to set or remove decorations from text

The value text-decoration:none; is used to remove underlines from links

Sample codes for text-decoration h2 { text-decoration: line-through; } p { text-decoration: underline; } a { text-decoration: none; }

Text Transformation

The text-transform property is used to specify uppercase and lowercase letters in a text

h2.uppercase { text-transform: uppercase; } p.lowercase { text-transform: lowercase; } h6.capitalize { text-transform: capitalize; }

The capitalize value is used to capitalize the first letter of each word

Text Spacing

There are 5 text spacing property

  1. Text Indentation
    • The text-indent property is used to specify the indentation of the first line of a text
      • p { text-indent: 30px; }
  2. Letter Spacing
    • The letter-spacing property is used to specify the space between the character in a text
      • p { letter-spacing: 1px; } h1 { letter-spacine: -2px; }
  3. Line Height
    • The line-height property is used to specify the space between lines
      • p.small { line-height: 1.5; }
  4. Word Spacing
    • The word-spacing property is used to specify the space between the words in a text
      • h1 { word-spacing: 8px; }
  5. White Space
    • The white-space property specifies how white-space inside an element is handled
      • h1 { white-space: nowrap; }
Text Shadow

Text shadow effect h1 { text-shadow: 5px 5px; } Text shadow effect using a different color h1 { text-shadow: 5px 5px green; } Text shadow effect with blur h1 { text-shadow: 3px 3px 7px green; }
